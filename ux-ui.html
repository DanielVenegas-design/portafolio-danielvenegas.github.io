Portafolio - Categoría

UX / UI

Diseño de interfaces enfocadas en la experiencia del usuario, combinando investigación, arquitectura de información y estética funcional para productos digitales responsivos.

Talent Trove

Este proyecto nació para solucionar la difusión y búsqueda de trabajo para diseñadores y animadores. Se desarrolló como un proyecto hipotético que conecta el talento creativo con oportunidades laborales a través de una plataforma intuitiva.

VCT Tracker Americas

Investigación y diseño de una página web responsiva para centralizar la información de los equipos de Valorant en la región de América. Se priorizó la organización de datos y la navegación fluida para los entusiastas de los eSports.

Barresama Inmobiliaria

Como parte del rediseño integral de identidad para Barresama, desarrollé su primera presencia digital oficial. El sitio web refleja la nueva imagen corporativa, enfocándose en la confianza y la claridad visual para sus clientes inmobiliarios.
